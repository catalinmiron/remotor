Remote Software Development (and more) Jobs

I firmly believe that the future of work is remote. Personally I'm based in the West of Spain (Extremadura) where there are very few job opportunities, but get by on remote consulting work.

The original idea was to gather links to Python development jobs from a whole lot of websites which carry ads for remote work. It has recently expanded to include other languages, and some other types of remote work opportunities which aren't tech-related.

This site is run on a totally voluntary basis and we aren't taking any kind of a cut or kickback from the job sites.

It's a work in progress so apologies in advance for the roughness around the edges. If you'd like to contribute to polishing it, feel free to fork the project which is up on Github here.


Good luck with the job-hunting!
